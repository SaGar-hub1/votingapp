About Voting App

Welcome to the Voting App! This application provides a seamless and interactive way for users to create and participate in polls. It is designed with simplicity and user experience in mind, ensuring that anyone can easily navigate and use the app.

Project Overview

The Voting App is a web-based application built using modern web technologies. It leverages the power of Spring Boot for the backend to handle all the server-side logic and data management. The frontend is crafted using HTML, CSS, and Bootstrap, ensuring a responsive and aesthetically pleasing user interface. MySQL is used as the database to store all the voting data securely.

Key Features

  • Create Polls: Users can create new polls by providing a question and multiple options for others to vote on.
  • Vote on Polls: Users can browse through existing polls and cast their votes on the options they prefer.
  • View Results: Poll results can be viewed in real-time, showing the number of votes each option has received.

Technologies Used

  • Backend: Spring Boot
  • Frontend: HTML, CSS, Bootstrap
  • Database: MySQL
